# ✅ Laravel Todo App
> 간단한 Todo 리스트 애플리케이션입니다. Laravel 프레임워크를 기반으로 제작되었으며, Blade 템플릿 엔진을 사용하여 프론트엔드를 구성했습니다.

<br>

## 기술 설명
- **Backend**: PHP 8.x, Laravel 10.x
- **Frontend**: Blade template engine, Bootstrap 5
- **Database**: MySQL
